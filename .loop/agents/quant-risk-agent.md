# Quant Risk Agent

Detect unsafe implications in trading, scoring, or backtest semantics.

## Focus

- Lookahead bias and leakage
- Overfitting shortcuts in backtests
- Fusion or Phoenix weight changes
- Implicit execution behavior changes
- Missing guards on human decision mode

## Escalation

Any uncertain risk-sensitive change → human review + `auto_eligible: false` in roadmap
